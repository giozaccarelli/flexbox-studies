A webpage that aims to test the uses of Flex-Box.

Properties that were used on this webpage

Display

It defines a flex container. It can be iniline or block depending on the given value. It also enables a flex context for all its direct children.

Example:

.container {
display: flex;
}

Flex-wrap

By default, flex items will try to fit into one line. We can change that and allow items to wrap as need with this property.

Example:

.container {
flex-wrap: nowrap | wrap | wrap-reverse;
}

Justify content

It defines the alignment along the main axis. It helps to distribute extra free space leftover when either all the flex items on a line are inflexible, or are flexible but have reached their maximum size.

Example

.container {
justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly
}
